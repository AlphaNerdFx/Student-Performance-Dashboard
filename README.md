# 📊 Student Performance and Risk Analysis Dashboard (Tableau)

An interactive Tableau project that analyzes student performance factors to identify success drivers and at risk behaviors. This dashboard combines behavioral effort metrics with environmental clustering to help academic advisors and department heads understand exactly why performance levels shift across the student population.

## 📊 Overview

This project uses the Student Performance Factors dataset to explore how 6,607 students are affected by socioeconomic backgrounds and daily habits. The dashboards are built with a high data to ink ratio, focusing on clear visual hierarchy and interactive parameters to turn raw data into an actionable student intervention tool.

## 🛠️ Features

### 🔹 Student Effort and Risk Dashboard (AE Dash)

- **Executive KPIs:** Real time tracking of total students, attendance, and a highlighted ten point four percent decline in exam scores.
- **Behavioral Matrix:** Interactive analysis of how motivation and tutoring sessions interact to drive total study hours of the student.
- **Risk Identification:** A specialized bar chart flagging at risk students in red based on low attendance and low study hours.
- **Performance Drivers:** An interactive area chart showing the ratio between sleep, attendance, and exam results.

### 🔹 Environmental Clustered Dashboard

- **Four Quadrant Analysis:** Deep dives into social setting, university setting, lifestyle, and learning environment.
- **Measure Switching:** A dynamic selector that allows the user to update all four quadrants to view attendance, previous scores, or current results.
- **Socioeconomic Variance:** Advanced box and whisker plots that analyze score distribution by family income, parental involvement, and education level.

## 📱 Interactive UI

- **Embedded Parameters:** Toggle demographic views between income, gender, and peer influence without switching tabs.
- **Dynamic Dashboard Navigation:** Seamlessly move between the behavioral effort view and the environmental clustering view.
- **Clean Design:** Strategic removal of gridlines and borders to maximize clarity and focus on teal performance highlights.

## 🔍 Preview

- **AE Dash:** Focuses on individual student effort and specific risk factors.
- **Clustered Dash:** Focuses on systemic environmental and socioeconomic drivers of performance.

## 🧰 Tools and Tech

- **Tableau:** Interactive dashboards, parameter actions, and statistical distribution plots.
- **Kaggle:** Dataset source and data preparation.
- **GitHub:** Documentation and version control.

## 🔹 How to Use

- Clone this repository to your local machine.
- Open the Tableau workbook (.twb / .twbx) in Tableau Desktop or Tableau Public.
- Explore the data by using the measure selectors and demographic parameters on each tab.
- Switch between the AE Dash and Clustered Dash to see different analytical perspectives.
